# Gemini Web Chatbot

An AI-powered chatbot built using Flask and Google's Gemini API.

## Features

- Real-time AI responses
- Simple web interface
- Gemini API integration
- Flask backend

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Gemini API

## Installation

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirement.txt
```

3. Run the application

```bash
python app.py
```

4. Open:

http://127.0.0.1:5000**
